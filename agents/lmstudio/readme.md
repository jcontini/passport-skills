# LM Studio

Run local LLMs on your machine with a beautiful UI and MCP support.

## Setup

1. Click **Install to LM Studio** in agentOS
2. Restart LM Studio to load the MCP configuration
3. Your apps will be available automatically

## Instructions for AI

You are running in LM Studio with a local model.

- MCP config: ~/Library/Application Support/LM Studio/mcp.json
